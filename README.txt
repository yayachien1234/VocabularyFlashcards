# VocabularyFlashcards

一個單字卡應用程式，幫助使用者記錄和學習單字。使用者可以新增單字、搜尋單字，並透過調整字型大小來方便查看單字列表。

## 功能
- **單字管理**：記錄單字的英文、中文解釋及詞性。
- **新增單字**：使用者可以在新增模式下新增新的單字卡片。
- **搜尋單字**：支援多條件搜尋，可根據不同條件篩選單字。
- **檔案操作**：
  - **新增**：清空現有資料，開始新的單字卡記錄。
  - **開啟舊檔**：讀取先前儲存的單字檔案。
  - **儲存**：將目前的單字卡資料存檔。
  - **另存新檔**：另存為新檔案。
  - **離開**：結束程式。
- **字型調整**：可以調整單字列表的字型大小，便於瀏覽。

## 使用說明
1. **新增單字**：進入新增模式，填寫單字、中文解釋和詞性後點擊「新增」按鈕。
2. **搜尋單字**：進入搜尋模式，選擇篩選條件後進行搜尋。
3. **檔案操作**：使用「檔案」選單執行新增、開啟、儲存、另存新檔及離開操作。
4. **調整字型**：在「檢視」選單中選擇字型大小來調整單字列表的顯示。

## 功能
- **單字測驗**：開啟測驗頁面，隨機從單字卡中挑選單字進行測驗。
- **顯示單字詳情**：在測驗過程中，點擊按鈕可以顯示單字的中文解釋和詞性。
- **標記單字**：使用者可以標記或取消標記單字。
- **只顯示標記單字**：切換到只顯示被標記的單字列表模式。
- **清除標記**：清除所有單字的標記狀態。

## 使用說明
1. **開始測驗**：開啟測驗頁面，會隨機顯示一個單字。
2. **查看單字詳情**：點擊「查看」按鈕顯示該單字的中文解釋和詞性。
3. **標記單字**：點選標記選項，對單字進行標記或取消標記。
4. **只顯示標記單字**：啟用此功能後，列表中只會顯示已標記的單字。
5. **清除標記**：清除所有單字的標記。
